## Namespaces

Commom used namespaces:

- `l-`: Layouts
- `o-`: Objects
- `c-`: Components
- `js`: Javascript hooks
- `is-` | `has`: State classes
- `t1` | `s1`: Typography sizes
- `u-`: Utility classes
